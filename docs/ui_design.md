# UI 设计界面说明文档

本通过文档描述了《赛博防御塔防》项目的新增 UI 界面设计方案，包括开始菜单和游戏结束界面。

## 1. 开始菜单 (StartMenu)

### 1.1 界面布局
- **背景**: 展示游戏地图的模糊视图或特定的赛博风格概念图。
- **标题**: "Cyber Defense TD" (赛博防御塔防)
    - 字体: 粗体、大号、带有霓虹发光效果 (Cyan/Magenta)。
    - 位置: 屏幕顶部中央。
- **菜单选项**:
    - **开始游戏**: 点击后进入游戏主场景。
    - **退出游戏**: 点击后退出应用程序。
    - 按钮样式: 带有科技感的边框，悬停时发光或变色。
- **版本号**: 屏幕右下角显示当前版本 (e.g., v0.1.0)。

### 1.2 交互逻辑
- **Start Game**: 切换到 `GameLevel` 场景，重置 `GameManager` 数据。
- **Quit**: 调用 `get_tree().quit()`。

---

## 2. 游戏结束界面 (GameOverScreen)

### 2.1 界面布局
- **背景**: 半透明黑色遮罩 (Overlay)，覆盖在当前游戏画面之上。
- **标题**: "MISSION FAILED" (任务失败)
    - 字体: 大号、红色警告风格。
- **统计信息**:
    - "Waves Survived: X" (生存波数)
    - 其他统计 (可选): 击杀数、获得金钱。
- **操作按钮**:
    - **Retry (重试)**: 重新加载当前关卡。
    - **Main Menu (主菜单)**: 返回开始菜单。

### 2.2 交互逻辑
- **Retry**: 重置游戏状态，重新加载 `GameLevel`。
- **Main Menu**: 切换到 `StartMenu` 场景。

---

## 3. UI 风格指南 (Style Guide)

- **主色调**: 
    - Neon Blue (#00FFFF)
    - Neon Pink (#FF00FF)
    - Dark Background (#111111)
- **字体**: 使用 Godot 默认字体或项目中已有的像素/科技风格字体。
- **动效**: 按钮点击反馈，简单的淡入淡出效果。
